Make sure that all your community is contained within the highlighted units, and that it does not contain gaps. The units are census block groups, which are used by map drawers to draw voting districts.

{% trans "How to draw my community?" %}

{% blocktrans %} Select units on the map by clicking with your mouse. To deselect a unit, click it again. Watch the following video to learn more. {% endblocktrans %}

{% trans "What are the units?" %}

{% blocktrans %} The units are census block groups, which are used by map drawers to draw voting districts. {% endblocktrans %}

{% trans "Why am I unable to save my map?" %}

{% blocktrans %} Make sure that it does not contain gaps. Communities of Interest must be contiguous. {% endblocktrans %}

{% trans "Can I map outside of my state?" %}

{% blocktrans %} In order to map outside of your state, you need to add a new community within the other state. {% endblocktrans %}

{% trans "How large should my community be?" %}

{% blocktrans %} Your community can be as small as one unit or as large as 800 units. {% endblocktrans %}
